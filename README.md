# PG商城

基于react-router（remix）框架的ssr前端项目

## 技术栈

### 路由

**react-router**

react-router 作为官方推荐的路由库，V7版本采用framworker模式，优化了路由结构。并且通过loader hooks能降低白屏问题

### 组件库

**ant design**

antd以其完善的组件数量成为国内最主流的react组件库，能大幅提升项目开发速度

### 二维图表

**echart**
通过echart实现数据可视化，帮助用户能实时查看数据，了解商品情况

### 双端互通

**webscoket**
基于tcp的长连接应用层协议，是跨端实时交互必不可少的技术，借助websocket实现用户端与管理端的实时通信

### 状态管理

**zustand**
轻量的react状态管理库，使用率也在逐步上升

## 项目

启动

```
pnpm dev
```

打包

```
pnpm build
```
